# FULL AI STACK

This Skill provide information about how download, install full ai stack and how configure agents installed.

## When to Use This Skill
- When user ask explicitly install stack of:
  - Agents
  - Skills

- When user ask explicitly configure provider and model of agents.

## What does it serve for?

To know about what instructions is needed to install all stack of agents and/or skills it depends on:

- User ask install agents, see first step named "agents installation."
- User ask configure agents, see second step named "configuration of agents"
- User ask install skills, see tird step named "skills instalation"
- User ask install full stack, follow all steps bellow.

## FIRST STEP AGENTS INSTALLATION

To install agents, first you have to ask a question to user if he is sure to install agents. Once user confirm follow the instruction from here:
[agents-installation.md](references/agents-installation.md)



## SECOND STEP CONFIGURATION OF AGENTS
After to install agents, now you have to configure which provider and model are going to use. The instructions to follow are here:

[configure-agents.md](references/configure-agents.md)



## TIRD STEP SKILLS INSTALLATION

Currently there's no skills to install.
